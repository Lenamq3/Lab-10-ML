# Lab-10-ML

This lab explores the implementation of Support Vector Classification (SVC), which is one of the supervised machine learning algorithms used for classification problems. The main objective of this lab is to understand how SVC works and how it can be applied to classify data accurately using Python programming and machine learning libraries.

Throughout the notebook, several important machine learning steps are performed, starting with loading and preprocessing the dataset to prepare it for training. The data is then divided into training and testing sets in order to evaluate the performance of the model properly. After that, the SVC model is created and trained using the training data.

Once the model training process is completed, predictions are generated using the testing data. Different evaluation methods are then used to measure the model performance and accuracy. This lab helps in developing a better understanding of classification techniques and practical machine learning workflows using Scikit-learn.
